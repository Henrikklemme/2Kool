2KOOL STUDIO — GITHUB PAGES PACKAGE
=====================================
Built for: https://henrikklemme.github.io/2Kool/

UPLOAD
------
1. GitHub (henrikklemme account) -> New repository named exactly:
   2Kool  (capital K, Public)
2. Add file -> Upload files -> ALL 12 files -> Commit
3. Settings -> Pages -> Deploy from a branch -> main / (root) -> Save
4. Live at: https://henrikklemme.github.io/2Kool/

- *.js / *.css      All scripts and styles

FEATURES
--------
- Contact form posts to Formspree (https://formspree.io/f/mwvgqrpp)
  Sends name, email, website. Shows success/error feedback.

- 16 photos load from the live Lovable site — if that project is
  unpublished, those images will break.
- Favicon: custom terracotta asterisk on cream, relative path fix
